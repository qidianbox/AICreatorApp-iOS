# 对标App (WanSnap/丸拍AI) UI细节分析

## 从录屏中提取的关键UI细节

### 1. 首页/发现页
- **顶部导航**: "WanSnap" Logo + 积分显示 (⚡360)
- **分类Tab**: 首页(紫色高亮胶囊) | 美图 | 生活 | 萌宠 | 旅拍 | 更多...
  - 选中状态: 紫色渐变胶囊背景
  - 未选中: 白色文字 + 小图标
- **瀑布流卡片**:
  - 圆角矩形图片
  - 底部显示: 标题 + 用户头像 + 用户名 + 浏览量(如 20.28K 👁)
  - 部分卡片底部有小图预览条(3张缩略图)
- **底部导航**: 3个Tab
  - 左: 发现图标
  - 中: 紫色渐变圆形按钮(创作入口)
  - 右: 个人中心图标

### 2. 新人福利弹窗
- 半透明紫色背景弹窗
- 礼物图标 + "⚡360" 大字
- "新人福利积分赠送" 标题
- "欢迎使用丸拍AI，期待你的创作哦！" 副标题
- 紫粉渐变按钮 "立即领取"

### 3. 作品详情页
- **顶部导航**: 返回按钮(紫色圆形) + 标题 + 分享按钮
- **主图区域**: 大图展示 + 左上角"AI生成"标签
- **用户信息**: 头像 + 用户名
- **提示词区域**: 
  - 深色背景卡片
  - "提示词" 标签 + 复制按钮
  - 提示词文字 + "展开" 按钮
- **跟图列表**:
  - "跟图列表" 标题 + 粉色爱心
  - 用户头像 + 用户名 + 生成的图片
  - 右侧点赞数 + 爱心图标
- **底部按钮**: 紫粉渐变 "跟图生成" 按钮

### 4. 分类Tab切换
- 选中Tab有紫色渐变背景
- Tab带有小图标(美图-相框, 生活-日历, 萌宠-爪印, 旅拍-相机)

### 5. 颜色规范
- **主色**: 紫色渐变 (#a855f7 → #ec4899)
- **背景**: 深色 (#0D0D0D 或 #1a1a2e)
- **卡片背景**: rgba(255,255,255,0.05)
- **文字**: 白色主文字, 灰色次要文字
- **强调色**: 粉色爱心, 紫色按钮

### 6. 交互细节
- 卡片悬停有微妙的放大效果
- Tab切换有平滑动画
- 按钮有渐变发光效果
- 返回按钮是紫色圆形背景

4. **底部导航** - 中间创作按钮改为紫色渐变圆形
5. **新人福利弹窗** - 可选添加

- 标题文字: 白色粗体
- 用户信息: 头像(小圆形) + 用户名(灰色)
- 浏览量: 数字 + 眼睛图标

- 高度约60px

- 右Tab: 个人图标(线条风格)
- 选中状态: 图标填充或高亮
