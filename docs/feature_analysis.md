## 三、UI设计特点

### 1. 视觉风格
- **主色调**: 深色背景（#1A1A1A）+ 紫粉渐变强调色
- **风格**: 暗黑模式为主，科技感与时尚感结合
- **圆角**: 大圆角设计，按钮和卡片都采用圆角

### 2. 布局特点
- **瀑布流**: 双列不等高瀑布流展示作品
- **卡片式**: 内容以卡片形式呈现
- **底部导航**: 三Tab设计，中间创作按钮突出

### 3. 交互特点
- **渐变按钮**: 主要CTA按钮使用紫粉渐变
- **底部弹窗**: 分享等操作使用底部弹出面板
- **滑动切换**: 图片对比支持左右滑动

## 四、核心业务流程

### 1. 图片生成流程
1. 用户选择模板/进入创作页面
2. 上传照片（相册/拍照）
3. 选择风格/模板
4. AI生成图片
5. 预览/保存/分享

### 2. 跟图生成流程
1. 浏览发现页作品
2. 点击进入作品详情
3. 查看提示词
4. 点击"跟图生成"
5. 上传自己的照片
6. 生成同风格图片
